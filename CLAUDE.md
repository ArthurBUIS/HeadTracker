# CLAUDE.md — VideoStitcher

## Standing rules

- Work on the **main** branch, unless you are told to work on another specific branch
- **Read the relevant `progress.md` before opening source files.** These docs are the primary map of the repo and exist to save tokens — only read source when the doc is insufficient or you're editing that code.
- When you change code in a directory, update that directory's `progress.md` in the same change. If the change alters cross-module flow, also update the root `/progress.md`.
- Keep every `progress.md` **dense and skimmable**: tables over prose, no filler, factual claims only.
- Cross-references between docs must be valid and bidirectional.
